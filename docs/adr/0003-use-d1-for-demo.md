# ADR-0003: 데이터베이스로 Cloudflare D1 채택 (데모)

- 상태: 채택
- 일자: 2026-06-23

## 맥락

데모용 저장소가 필요하다. Workers 와 통합이 쉽고 무료 한도 내에서 운영 가능해야 한다.

## 결정

데모 단계에서는 Cloudflare D1(SQLite 기반)을 사용한다.

## 근거

- Workers 와 바인딩으로 직접 통합되어 추가 인프라가 없다.
- SQLite 호환이라 스키마/쿼리가 단순하고 로컬(`wrangler ... --local`)에서 동일하게 동작.
- 데모 규모의 읽기 위주 트래픽에 충분.

## 트레이드오프

- 쓰기 동시성·대용량에는 한계 → 호출 액션은 `200/409` 동시성 처리로 보완(부하 테스트에 반영).
- 운영 규모에서는 MySQL/Postgres 이전이 필요할 수 있음 → ADR-0006, scaling-strategy 3단계.

## 대안

- 외부 관리형 DB(Neon/Upstash 등): 가능하나 데모 단계에선 통합·비용 측면에서 과함.
