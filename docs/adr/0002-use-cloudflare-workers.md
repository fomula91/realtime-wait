# ADR-0002: 백엔드로 Cloudflare Workers 채택

- 상태: 채택
- 일자: 2026-06-23

## 맥락

포트폴리오용 무료 데모를 안정적으로 운영해야 한다. 실제 운영 구조(AWS ECS 등)는 비용과 관리 부담이 크다.

## 결정

백엔드 API 를 Cloudflare Workers(Hono 프레임워크)로 구현한다.

## 근거

- 무료 한도 안에서 데모 유지가 쉽다.
- 요청 수 기반 과금/한도라 `사용자 수 × 폴링 주기` 로 부하·비용을 예측 가능(ADR-0004 와 시너지).
- 서버리스라 인프라 운영 부담이 거의 없다.
- D1(ADR-0003)과 자연스럽게 통합된다.

## 트레이드오프

- 장기 연결(WebSocket/SSE)·CPU 장시간 작업에는 제약이 있다 → 데모에서 polling 선택의 한 배경.
- 실제 대규모 운영 구조(컨테이너 오토스케일 등)와는 형태가 다르다 → 확장안은 ADR-0005/0006.

## 대안

- AWS ECS / 직접 서버: 운영 구조에 가깝지만 비용·관리 부담.
- 다른 서버리스(Lambda 등): 가능하나 D1·엣지 통합 이점이 적다.
